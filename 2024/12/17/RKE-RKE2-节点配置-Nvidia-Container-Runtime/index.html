GPU Operator

NVIDIA GPU Operator 是 NVIDIA 提供的一个 Kubernetes Operator，用于自动部署、配置和管理 Kubernetes 集群中的 GPU 软件栈。

它遵循 Kubernetes Operator 模式，通过一个 ClusterPolicy CR 来自动安装和维护 GPU 相关组件，无需手动安装和配置每个节点。
