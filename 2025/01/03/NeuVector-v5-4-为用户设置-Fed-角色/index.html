NeuVector v5.4 为用户设置 Fed 角色

NeuVector 从 v5.4 版本起，通过 Rancher 登录到 NeuVector 的用户无法在 NeuVector 被设置为 Fed 角色，需要在 Rancher 中设置对应的角色，从而映射到 NeuVector 中。

对于 NV 在 Primary 集群中的 SSO 映射：

Global Role：

  • get/nv-perm.all-permissions → fedReader
  • */nv-perm.all-permissions → fedAdmin

Cluster Role：

  • get/nv-perm.all-permissions → reader
  • */nv-perm.all-permissions → admin
  • get/nv-perm.all-permissions,nv-perm.fed → fedReader
  • */nv-perm.all-permissions,nv-perm.fed → fedAdmin

Project Role：没有项目角色会映射到 fedReader/fedAdmin 角色。

如果不是 Primary NV 集群，映射的 fedReader 角色会自动降级为 reader 角色，fedAdmin 角色会自动降级为 admin 角色。

Global Role
